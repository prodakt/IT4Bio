## Course Description
This course introduces biology and biotechnology students to practical applications of the R programming language, AI-assisted coding, and foundational IT tools used for biological data analysis. It covers programming basics, biological data import, visualization, bioinformatics workflows and code repositories.

## Learning Outcomes
After completing the course, students will be able to:
- understand R fundamentals and write clean, readable code,
- import and process biological data (FASTA, BLAST, Ensembl, MSA),
- create high-quality, publication-ready visualizations,
- use basic bioinformatics tools in R (e.g., rBLAST, rentrez),
- apply AI tools (ChatGPT, Colab, AI Studio) to assist in coding and analysis,
- write functions, use loops, and work with CRAN, Bioconductor and GitHub.

## Course Outline

### 1. Basics of R and AI-assisted programming
- Working environment: RGui, RStudio
- Console vs scripts, data structures, clean code
- Data frame operations, simple graphics
- Introduction to AI tools: ChatGPT, Colab, AI Studio
- How to prompt AI, verify code and accelerate workflows

### 2. Importing biological data and visualizing statistics
- BLAST and rBLAST
- Ensembl genome statistics
- Importing FASTA, PSA, MSA
- Basic visualization of biological datasets

### 3. Advanced graphics: ggplot2, plotly and high-quality export
- Publication-ready graphics
- Colors, themes, faceting
- Export to TIFF, PNG, PDF, BMP

### 4. Bioinformatics workflows in R
- rBLAST
- rentrez (NCBI API)
- MSA and sequence analysis

### 5. Functions, loops, packages and repositories
- Writing functions
- Loops, apply-family
- CRAN, Bioconductor, GitHub
- Documentation and reproducibility

### 6. Optional extended modules
- Introduction to R Markdown and Quarto
- Basics of Git and GitHub
- Simple bioinformatics pipelines
