# IT4Bio — Technologie informatyczne dla biologów i biotechnologów

## Opis kursu
Kurs wprowadza studentów biologii i biotechnologii w praktyczne wykorzystanie języka R, narzędzi AI oraz podstawowych technik informatycznych w pracy z danymi biologicznymi. Obejmuje podstawy programowania, import danych z baz biologicznych, wizualizację, analizy bioinformatyczne oraz pracę z repozytoriami kodu.

## Cele kursu
Po ukończeniu kursu student:
- rozumie podstawy języka R i potrafi pisać czytelny, poprawny kod,
- potrafi importować i przetwarzać dane biologiczne (FASTA, BLAST, Ensembl, MSA),
- umie generować wykresy wysokiej jakości i eksportować je do standardów publikacyjnych,
- zna podstawowe narzędzia bioinformatyczne dostępne w R (np. rBLAST, rentrez),
- umie wykorzystywać AI (ChatGPT, Colab, AI Studio) w tworzeniu, analizowaniu i poprawie kodu,
- potrafi tworzyć funkcje, pętle oraz korzystać z CRAN, Bioconductor i GitHuba.

## Program kursu

### 1. Podstawy R i wykorzystanie AI w programowaniu
- Środowisko pracy: RGui, RStudio
- Konsola, skrypty, struktury danych, czystość kodu
- Operacje na ramkach danych, generowanie podstawowych wykresów
- Wprowadzenie do narzędzi AI: ChatGPT, Colab, AI Studio
- Tworzenie i weryfikacja kodu z pomocą AI

### 2. Import danych z biologicznych baz i wizualizacja statystyk
- BLAST i rBLAST
- Ensembl i statystyki genomowe
- Import FASTA, PSA, MSA
- Podstawowa wizualizacja danych biologicznych

### 3. Zaawansowana grafika w R: ggplot2, plotly i eksport wysokiej jakości
- Tworzenie wykresów publikacyjnych
- Kolorystyka, style, faceting
- Eksport do TIFF, PNG, PDF, BMP

### 4. Bioinformatyczne analizy w R
- rBLAST
- rentrez (NCBI API)
- MSA i analiza sekwencji

### 5. Funkcje, pętle, pakiety i repozytoria
- Tworzenie funkcji w R
- Pętle, apply-family
- CRAN, Bioconductor, GitHub
- Dokumentacja i reproducibility

### 6. Moduły rozszerzone (opcjonalne)
- Wprowadzenie do R Markdown i Quarto
- Podstawy Git i GitHub
- Proste workflowy bioinformatyczne
